Frequently Asked

Questions

How do I use commands?

Open console by using tilde(~) and type "help" or go to commands to see commands.

When's the next update?

No one knows, the game being developed slowly and it's not the biggest priority in the dev's lives.

How do I turn off the hud?

Pressing F1 will disable the hud, if you want your guns gone as well for screenshots or similar I would recommend using viewmodel_offset

How do I switch maps?

In a private server where you have permissions, open the console by using tilde(~) and type "maps" to see all maps, and then type "map mapname" to change the map

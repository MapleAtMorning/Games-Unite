Questions

When will the game come out?

No one knows, the game being developed is not the biggest priority in the dev's lifes.
Please leave them alone.

How do I turn off the hud?

Pressing F1 will disable the hud, if you want your guns gone as well for screenshots or similar I would recommend using viewmodel_offset

How do I use commands?

Open console by using tilde(~) and type "help" or go to commands to see commands.

Help! I've been banned??

If you believe you've been falsely banned in-game or from the Discord server please submit an appeal in the Support Server

How do I switch maps?

Open console by using tilde(~) and type "maps" to see all maps, and then type "map mapname" to change the map
